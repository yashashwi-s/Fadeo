# Marketing

Promo reel for Fadeo.

- `fadeo-reel-landscape.mp4` — 1920x1080, 22s, 60fps. The launch reel: the real Now pane
  reacting to app switches (Deep Work plays, Reading pauses, silence), then a fast tour of
  the panes. Built with Remotion. Cut to a 120 BPM grid; the soundtrack hard-pauses when the
  app shows "paused" and slams back on the drop.
- `fadeo-reel-720.mp4` — 1280x720 copy for lighter uploads.
- `fadeo-reel-audio.m4a` — the edited audio bed alone.
